# Runch

A launcher for source code as interpreter.

## Installation

    gem install runch

## Support Languages

  * C
  * C++ (extentions: cc, cp, cpp, cxx)
  * Java

## Requirements

  * C
    * gcc
  * C++
    * g++
  * Java
    * javac

## Usage

They look like an interpreter!

### C

    runch hello.c

### C++

    runch hello.cc

### Java

    runch Hello.java
